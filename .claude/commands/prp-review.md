# Review Existing PRP

Analyze a PRP for completeness, clarity, and quality.

## Process

1. **Structure check** — All sections present (Context, State, Requirements, Constraints, Success Criteria)
2. **Requirements quality** — Each is Specific, Testable, Independent, Valuable, Estimable, Small
3. **Constraints validation** — Timeline realistic, tech compatible, dependencies clear
4. **Success criteria** — At least 3, measurable, covers technical + business + quality
5. **Risk identification** — Scope creep, unrealistic timeline, dependencies, technical risks

## Output: Score 1-10 + issues by severity (Critical/Important/Minor) + recommendations
